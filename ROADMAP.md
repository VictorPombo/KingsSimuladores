# Roadmap

- [ ] **Phase 1: Architecture & Setup**
  Repositório, Supabase models (produtos + `cnpj_emitente`), rotas internas da API.

- [ ] **Phase 2: Core da Loja + Catálogo**
  Layout base, design system, produtos, carrinho, filtros.

- [ ] **Phase 3: Integrações Críticas**
  Mercado Pago, Frete, webhook Olist, lógica de emissão de NF (simples e dupla).

- [ ] **Phase 4: Marketplace de Usados**
  Área vendedor, aprovação admin, comissionamento automático.

- [ ] **Phase 5: Otimização Comercial**
  Upsell, cupons, recuperação de carrinho.

- [ ] **Phase 6: Painéis e Relatórios**
  Analytics administrativo por CNPJ, painel do cliente.

- [ ] **Phase 7: Testes E2E & Go Live**
  Fluxos reais de compra mistos, migração de dados.
